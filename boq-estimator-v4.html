Project Setup

Choose the project type, market and building code. The app then adapts the floor and room structure to that scope instead of forcing one single template.

Scope Selection

The user must choose which packages are included in the BOQ and costing. Room-wise specs then drive the quantities under those selections.

Floors and Rooms

Use the auto-generated layout, then adjust any floor or room in detail. Quick dropdowns below help navigate large projects.

Subcontractor Packages

Generate scope-wise package PDFs only when needed. Each package includes related materials, quantities and issue-ready scope notes for that trade.

Value Engineering

Keep value engineering as a separate workflow inside the app so users can review practical cost and coordination opportunities without cluttering the main BOQ table.

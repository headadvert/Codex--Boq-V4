# BOQ Estimator v4

Static GitHub Pages build for the v4 BOQ and Cost suite.

Files:
- `index.html` landing page
- `access.html` lead and access flow
- `boq-estimator-v4.html` main BOQ route
- `cost-estimator-v4.html` pricing and benchmark route
- `privacy.html` browser-storage privacy note
- `boq-estimator.html` and `cost-estimator.html` compatibility redirects
- `assets/app.css` shared styling
- `assets/app.js` shared behavior, data model, BOQ generation, pricing, benchmark and PDF export

This build uses browser local storage and session storage only.

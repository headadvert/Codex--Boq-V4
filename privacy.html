Privacy

This static GitHub Pages version stores lead details, access state and project data in the browser local storage and session storage on the user's device. It does not include a backend database in this repo.

Lead capture: name, email, phone, role and purpose are stored locally so the access flow feels similar to v3.
Project data: project setup, floors, rooms, selected scopes and generated BOQ data are stored locally in the browser for convenience.
Pro code flow: the default static code example is only for local verification and demonstration until you replace it with your own production method.
